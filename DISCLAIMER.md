# Medical and Data Disclaimer
## For Research Purposes Only
This project is a proof-of-concept for a Bachelor's project. It is intended strictly for academic research.
## Data Usage and Attribution
This research utilizes the MIMIC-IV database.  
**Citation:** If you use the data processing scripts in this repository, please cite both this repository and the original MIMIC-IV paper:  
_Johnson, A., Bulgarelli, L., Pollard, T., Gow, B., Moody, B., Horng, S., Celi, L. A., & Mark, R. (2024). MIMIC-IV (version 3.1). PhysioNet. RRID:SCR_007345._  
Additionally, as per indicated on the physionet.org page dedicated to the MIMIC-IV database, the dataset of patients are de-identified.
